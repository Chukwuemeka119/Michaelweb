Simple use of HTML

HTML is the standard mark-up language for Web Pages. With HTML, you can create your own Web Page. HTML is used with the combination of CSS, JavaScript to create beautiful, functional and exiciting Web Pages.

If you wish to test this examples out yourself, follow this instruction;

For Pc:

You will be needing a browser and any text editor such as Notepad( an inbuilt text editor), atom, Notepad++, Vs Code, e.t.c.
After getting a text editor, you need to copy the code, and paste it in your text editor. You then need to save it with an extention of .html File. After you are done saving it, you need to reopen the file using any broswer.

For Phones
download a text ediotor such as 'treb edit' and follow the instruction above.
